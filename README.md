# News Bot

Telegram-бот, который собирает популярные новости по теме и публикует в канал

Новости берутся с сайта sports.ru по выбранному тегу

Бот собирает последние новости по тегу и выбирает из них самые обсуждаемые

## Настройка

Создайте файл `.env` в корне проекта

```shell
TAG_ID=123456 # ID тега
NEWS_LIMIT_ALL=50 # количество последних новостей
NEWS_LIMIT_TOP=20 # количество топовых
TELEGRAM_TOKEN=54321:ABCDEF # токен telegram-бота
TELEGRAM_CHANNEL_ID=-1003925442669 # ID канала
TELEGRAM_PROXY=socks5://user:password@0.0.0.0:00 # прокси для telegram
DB=news.db # файл для хранения опубликованных новостей
```

## Запуск

Скомпилируйте

```bash
go build -o news-bot ./cmd/cli
```

Если по умолчанию не включено CGO, то можно включить принудительно

```bash
CGO_ENABLED=1 go build -o news-bot ./cmd/cli
```

Запустить
```bash
./news-bot
```

Запуск скрипта можно добавить в cron, чтоб он автоматически публиковал новые посты

## Настройка бота в Telegram

Создайте нового бота в Telegram через @BotFather

Создайте новый канал в Telegram

Добавьте бота в админы канала

Чтоб узнать ID канала можно создать любое сообщение и переслать его боту @getidsbot Он покажет ID канала
